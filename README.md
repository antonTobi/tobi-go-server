# tobi-go-server

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (comes with Node.js)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/antonTobi/tobi-go-server.git
cd tobi-go-server
```

2. Install dependencies:
```bash
npm install
```

### Running the App

Start the development server:
```bash
npm run dev
```

This runs `netlify dev` which provides local testing with the same routing as production.

### Deployment

Deploy to Netlify:
```bash
npm run deploy
```

Or use Netlify's Git integration by connecting your repository at [netlify.com](https://app.netlify.com).
